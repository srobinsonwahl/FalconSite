FalconSite

By utilizing publicly available data harvested via internet crawlers from a third-party, Shodan, FalconSite accesses up-to-date data on all IP addresses connected to the internet, and does so in an entirely legal, transparent, and efficient manner. The data maps all open ports, or internet-facing ports, that are actively listening for packets to be sent over the internet to them. While open ports are not inherently unsafe, an open port is like leaving your house door unlocked, but keeping the door closed. If someone wants to get in, and the system is not set up to control these infiltrators, they surely will get in.
